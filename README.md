# Depth Map to Mesh
Convert depth map to 3D mesh in .obj format.

A re-implementation of https://github.com/CapObvios/Depth-Map-Visualizer with additional functions.

## Requirements
- Python 3.2+
- OpenCV
- Numpy

## Usage
```
python depthmap2mesh.py [DEPTH_MAP]
```

See more options:
```
python depthmap2mesh.py -h
```
